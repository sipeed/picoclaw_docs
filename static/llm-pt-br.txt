# PicoClaw

> PicoClaw é um assistente pessoal de IA ultra-leve escrito em Go. Roda em hardware de $10 com menos de 10MB de RAM e boot em menos de 1 segundo. Suporta 14 canais de chat (Telegram, Discord, Slack, etc.), 19+ provedores de LLM, e funcionalidades avançadas como hooks, steering, MCP e tarefas assíncronas.

Lembre-se:

- PicoClaw é um binário Go único e autocontido. Arquiteturas suportadas: x86_64, ARM64, ARMv6, RISC-V, LoongArch e MIPS.
- Toda configuração fica em `~/.picoclaw/config.json`. Qualquer campo pode ser sobrescrito via variável de ambiente `PICOCLAW_<SECAO>_<CHAVE>`.
- O campo `model_list` configura provedores LLM usando o formato `protocolo/modelo` (ex: `openai/gpt-4o`, `anthropic/claude-sonnet-4-20250514`).
- Ferramentas web (`web_fetch`) fazem HTTP GET simples sem executar JavaScript. Fallback via Firecrawl se configurado.
- O modo `picoclaw gateway` inicia o servidor HTTP para receber mensagens dos canais de chat.
- O modo `picoclaw onboard` inicializa o workspace e gera a configuração padrão.
- Segurança: por padrão, o sandbox restringe acesso a arquivos e comandos ao workspace. Mais de 30 padrões de comandos perigosos são bloqueados.

## Canais de Chat

- [Visao Geral](https://docs.picoclaw.io/channels): Lista dos 14 canais suportados com nivel de dificuldade e notas
- [Telegram](https://docs.picoclaw.io/channels/telegram): Canal recomendado. Facil configuracao, transcricao de voz com Groq Whisper, suporte a proxy
- [Discord](https://docs.picoclaw.io/channels/discord): Bot token + intents, gatilho de grupo, links de mensagem
- [Slack](https://docs.picoclaw.io/channels/slack): Socket Mode sem IP publico, indicador de digitacao, mensagens placeholder
- [QQ](https://docs.picoclaw.io/channels/qq): API oficial QQ Bot (AppID + AppSecret), modo sandbox para testes
- [DingTalk](https://docs.picoclaw.io/channels/dingtalk): Stream Mode via WebSocket, chat privado e em grupo
- [WeCom Bot](https://docs.picoclaw.io/channels/wecom-bot): Chat em grupo via webhook, opcao mais simples do WeCom
- [WeCom App](https://docs.picoclaw.io/channels/wecom-app): App auto-construido, chat privado e mensagens proativas
- [WeCom AI Bot](https://docs.picoclaw.io/channels/wecom-aibot): Bot IA oficial com protocolo streaming pull
- [Feishu](https://docs.picoclaw.io/channels/feishu): Plataforma empresarial ByteDance, modo WebSocket, cards interativos
- [LINE](https://docs.picoclaw.io/channels/line): Webhook HTTPS obrigatorio (ngrok/tunel), citacao de mensagens
- [OneBot](https://docs.picoclaw.io/channels/onebot): Compativel com NapCat e Go-CQHTTP, servidor WebSocket reverso
- [Matrix](https://docs.picoclaw.io/channels/matrix): Protocolo aberto e descentralizado, indicador de digitacao, suporte a midia
- [MaixCAM](https://docs.picoclaw.io/channels/maixcam): Protocolo TCP socket para cameras IA de borda, eventos de deteccao de pessoa
- [WeChat](https://docs.picoclaw.io/channels/weixin): Conta pessoal via API iLink com autenticacao por QR code
